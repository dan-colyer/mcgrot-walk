# Shopfront wishlist — D4.1/W2 handmade pipeline

151 frontage buildings still lack a real photographed identity (a borrowed
donor image, a generic AI rendition, or nothing at all) despite having a known
real business name. Sorted by visibility priority — near a real-photo building
or a street-junction corner first, since that's where the contrast with a
genuine photo is most visible.

## Drop-and-run

1. Find a reference photo of the real shop (Street View, the business's own
   site, a review site).
2. Paste it into ChatGPT with the prompt from that row's table below.
3. Save the result as the row's **target filename** under
   `assets/shopfronts/handmade/`.
4. `node scripts/ingest-handmade.mjs` — validates, grades, stamps, rebuilds the
   atlas. Safe to run after every single drop.
